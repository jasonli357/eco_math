要证明CES效用函数$u(x_1, x_2) = (x_1^\rho + x_2^\rho)^{\frac{1}{\rho}}$是严格拟凹函数，我们可以使用加边海森矩阵（Bordered Hessian Matrix）的方法。对于一个二元函数，如果其加边海森矩阵的行列式大于零，那么该函数就是严格拟凹的。

通常，CES效用函数的参数 ρ 被限制在 ρ ≤ 1 且 ρ ≠ 0 的范围内，同时消费量$x_1, x_2$为正。

### 1. 定义和条件

一个二元可微函数$u(x_1, x_2)$是严格拟凹的，如果满足以下条件：
其加边海森矩阵的行列式为正。

加边海森矩阵定义如下：
$H_B = \begin{pmatrix} 0 & u_1 & u_2 \\ u_1 & u_{11} & u_{12} \\ u_2 & u_{21} & u_{22} \end{pmatrix}$
其中，$u_i = \frac{\partial u}{\partial x_i}$是一阶偏导数，$u_{ij} = \frac{\partial^2 u}{\partial x_i \partial x_j}$是二阶偏导数。

我们需要证明 $det(H_B) > 0$。

### 2. 计算一阶偏导数

首先，我们计算效用函数的一阶偏导数：
$u_1 = \frac{\partial u}{\partial x_1} = \frac{1}{\rho} (x_1^\rho + x_2^\rho)^{\frac{1}{\rho} - 1} \cdot (\rho x_1^{\rho - 1}) = (x_1^\rho + x_2^\rho)^{\frac{1-\rho}{\rho}} \cdot x_1^{\rho - 1}$

通过对称性：
$u_2 = \frac{\partial u}{\partial x_2} = (x_1^\rho + x_2^\rho)^{\frac{1-\rho}{\rho}} \cdot x_2^{\rho - 1}$

为了简化表达，我们令$A = (x_1^\rho + x_2^\rho)$。
$u_1 = A^{\frac{1-\rho}{\rho}} \cdot x_1^{\rho - 1}$
$u_2 = A^{\frac{1-\rho}{\rho}} \cdot x_2^{\rho - 1}$

### 3. 计算二阶偏导数

接下来，计算二阶偏导数：
$u_{11} = \frac{\partial u_1}{\partial x_1} = (\rho - 1) x_1^{\rho - 2} A^{\frac{1-\rho}{\rho}} - (\rho - 1) x_1^{2\rho - 2} A^{\frac{1-2\rho}{\rho}}$
$u_{11} = (\rho - 1) x_1^{\rho - 2} A^{\frac{1-2\rho}{\rho}} (A - x_1^\rho) = (\rho - 1) x_1^{\rho - 2} x_2^\rho A^{\frac{1-2\rho}{\rho}}$

通过对称性：
$u_{22} = (\rho - 1) x_2^{\rho - 2} x_1^\rho A^{\frac{1-2\rho}{\rho}}$

以及交叉偏导数：
$u_{12} = \frac{\partial u_1}{\partial x_2} = (1 - \rho) x_1^{\rho - 1} x_2^{\rho - 1} A^{\frac{1-2\rho}{\rho}}$
根据杨氏定理（Young's Theorem），$u_{21} = u_{12}$。

### 4. 计算加边海森矩阵的行列式

加边海森矩阵的行列式为：
$det(H_B) = -u_1(u_1 u_{22} - u_2 u_{12}) + u_2(u_1 u_{12} - u_2 u_{11})$
$det(H_B) = -u_1^2 u_{22} + 2u_1 u_2 u_{12} - u_2^2 u_{11}$

我们将之前计算出的偏导数代入：
$u_1^2 = A^{\frac{2(1-\rho)}{\rho}} x_1^{2(\rho - 1)}$
$u_2^2 = A^{\frac{2(1-\rho)}{\rho}} x_2^{2(\rho - 1)}$
$u_1 u_2 = A^{\frac{2(1-\rho)}{\rho}} (x_1 x_2)^{\rho - 1}$

代入行列式表达式并提取公因式$(1-\rho)A^{\frac{2(1-\rho)}{\rho}}A^{\frac{1-2\rho}{\rho}} = (1-\rho)A^{\frac{3-4\rho}{\rho}}$：
$det(H_B) = (1-\rho)A^{\frac{3-4\rho}{\rho}} [x_1^{2\rho-2}x_2^{\rho-2}x_1^\rho + 2(x_1x_2)^{\rho-1}(x_1x_2)^{\rho-1} + x_2^{2\rho-2}x_1^{\rho-2}x_2^\rho]$

简化方括号内的表达式：
$[ ... ] = x_1^{3\rho-2}x_2^{\rho-2} + 2(x_1x_2)^{2\rho-2} + x_1^{\rho-2}x_2^{3\rho-2}$
提取公因式$(x_1x_2)^{\rho-2}$:
$[ ... ] = (x_1x_2)^{\rho-2} [x_1^{2\rho} + 2(x_1x_2)^{\rho} + x_2^{2\rho}]$
$[ ... ] = (x_1x_2)^{\rho-2} (x_1^\rho + x_2^\rho)^2 = (x_1x_2)^{\rho-2} A^2$

将简化后的结果代回行列式表达式：
$det(H_B) = (1-\rho)A^{\frac{3-4\rho}{\rho}} \cdot (x_1x_2)^{\rho-2} A^2$
$det(H_B) = (1-\rho) (x_1x_2)^{\rho-2} A^{\frac{3-4\rho}{\rho} + 2}$
$det(H_B) = (1-\rho) (x_1x_2)^{\rho-2} A^{\frac{3-2\rho}{\rho}}$
$det(H_B) = (1-\rho) (x_1x_2)^{\rho-2} (x_1^\rho + x_2^\rho)^{\frac{3-2\rho}{\rho}}$

### 5. 分析行列式的符号

为了使函数是严格拟凹的，我们需要$det(H_B) > 0$。我们来分析上式中每一项的符号（假设$x_1, x_2 > 0$）：
*  $(x_1x_2)^{\rho-2}$恒为正。
*  $(x_1^\rho + x_2^\rho)$恒为正，因此$(x_1^\rho + x_2^\rho)^{\frac{3-2\rho}{\rho}}$也为正。

因此，行列式的符号完全取决于$(1-\rho)$这一项。
要使$det(H_B) > 0$，必须满足：
$1 - \rho > 0 \implies \rho < 1$

### 结论

CES效用函数$u(x_1, x_2) = (x_1^\rho + x_2^\rho)^{\frac{1}{\rho}}$的加边海森矩阵行列式为正的条件是$\rho < 1$。因此，当$\rho < 1$时，该CES效用函数是严格拟凹函数。这对应于经济学中消费者偏好为严格凸（indifference curves are strictly convex）的情况。
